# Example: Form Submission

This example demonstrates a multi-field form submission workflow.

## Workflow Overview
1. Navigate to form page
2. Fill out personal details (Name, Email)
3. Select options (Checkboxes, Radio buttons)
4. Select from dropdown
5. Submit form

## Files
- `workflow.json`: The analyzed action sequence.
- `generated_script.py`: The generated Selenium automation script.
