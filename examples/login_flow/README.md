# Example: Login Flow

This example demonstrates a simple login workflow captured and automated with ShowOnce.

## Workflow Overview
1. Navigate to login page
2. Enter username
3. Enter password
4. Click Login button

## Files
- `workflow.json`: The analyzed action sequence.
- `generated_script.py`: The generated Playwright automation script.

## How to run
1. Ensure dependencies are installed: `pip install playwright`
2. Run playwright install: `playwright install`
3. Execute the script: `python generated_script.py`
